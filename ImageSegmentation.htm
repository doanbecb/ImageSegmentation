K-means biểu diễn cc cụm bởi cc trọng tm của cc đối tượng trong cụm đ. Thuật ton K-means chi tiết được trnh by:

BEGIN

1.     Nhập n đối tượng dữ liệu

2.     Nhập k cụm dữ liệu

3.     MSE = +∞;

4.     For I = 1 to k do mi = Xi+(i-1)*[n/k]; % khởi tạo k trọng tm

5.     Do {

6.     OldMSE = MSE;

7.     MSE = 0;

8.     For j = 1 to k do

9.     { m[j] = 0; n[j] = 0 }

10.                        End for

11.                        For I = 1 to n do

12.                        For j = 1 to k do

13.                        D2(x[i]; m[j]); % khoảng cch Euclide bnh phương

14.                        End for

15.                        %Tm trọng tm gần nhất m[h] tới X[i]

16.                        m[h] = m[h] + X[i] ; n[h] = n[h] + 1;

17.                        MSE = MSE + D2(x[i]; m[j]);

18.                        End for

19.                        n[j] = max(n[j], 1); m[j] = m[j]/n[j];

20.                        MSE = MSE;

21.                        } while(MSE <= OldMSE);

END

Trong đ :

-         MSE : Sai số bnh phương trung bnh hay l hm tiu chuẩn.

-         (x[i]; m[j]): khoảng cch Euclide từ đối tượng thứ i tới trọng tm j.

-         OldMSE m[j], n[j]: biến tạm lưu gi trị cho trạng thi trung gian cho cc biến tương ứng.
